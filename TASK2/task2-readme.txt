Task 2 - Movie Info Fetcher

คำอธิบาย:
โปรแกรมนี้รับชื่อภาพยนตร์จากผู้ใช้ และใช้ OMDb API เพื่อดึงข้อมูลภาพยนตร์ รวมถึงโปสเตอร์ หากมีโปสเตอร์อยู่ โปรแกรมจะดาวน์โหลดและบันทึกไว้ในเครื่องของผู้ใช้ โดยตั้งชื่อไฟล์ตามรูปแบบ <ชื่อภาพยนตร์>_poster_<รหัสนักศึกษา>.jpg

วิธีการรัน:
1. เปิดไฟล์ task2.py ด้วย Python 3 
2. ป้อนชื่อภาพยนตร์ที่ต้องการ เช่น Titanic, The space between us
3. โปรแกรมจะแสดงข้อมูลภาพยนตร์และบันทึกโปสเตอร์ใน working directory เดียวกับTask2.py
