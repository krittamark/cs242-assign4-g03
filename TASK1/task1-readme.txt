Task 1 - Project Gutenberg Top 100 EBooks Scraper

คำอธิบาย:
โปรแกรมนี้ดึงข้อมูลจากเว็บไซต์ **Project Gutenberg** และแสดงรายการ **Top 100 EBooks yesterday** พร้อมหมายเลขไฟล์และรายชื่อหนังสือ 
โดยใช้ **BeautifulSoup** เพื่อแปลง HTML และดึงข้อมูลที่เกี่ยวข้อง

วิธีการรัน:
1. เปิดไฟล์ `task1.py` ด้วย Python 3
2. โปรแกรมจะทำการร้องขอข้อมูลจากเว็บไซต์ Project Gutenberg
3. แสดงผลลัพธ์ต่อไปนี้:
   - ลิงก์ทั้งหมดจากหน้าเว็บ (แสดงตัวอย่าง 30 ลิงก์แรก)
   - หมายเลขไฟล์ eBooks 100 อันดับแรก
   - ข้อความจากหน้าเว็บ (2000 ตัวอักษรแรก)
   - รายชื่อหนังสือ Top 100 พร้อมชื่อผู้แต่ง

โมดูลที่ต้องติดตั้ง:
- requests 
- re 
- BeautifulSoup` (bs4)
